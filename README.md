# Supply Chain Analytics

## Project Overview

This project analyzes supply chain performance by tracking delivery efficiency, fulfillment rates, and revenue impact using SQL, PostgreSQL (Supabase), Anomaly AI, and n8n.

## Architecture

Source Data (Gmail)
→ n8n Workflow
→ PostgreSQL (Supabase)
→ Anomaly AI
→ Analytics Dashboard

## Key Insights

- Revenue loss from undelivered quantities: ₹34.13 Million
- 203,158 undelivered units
- 8,683 shorted order lines
- Average delivery delay: 1.69 days
- Lowest In-Full Rate: Food (65.87%)
- Highest OTIF discrepancy: Coolblue, Vadodara (673 lines)
